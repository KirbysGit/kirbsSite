Project Overview

SecureScape is a portable, smart security system designed for remote locations without internet access. Utilizing ESP32-CAM microcontrollers equipped with infrared sensors, our system detects motion in real-time, processes images locally using a custom-built, lightweight image analysis algorithm for person detection, and communicates alerts through a central control unit. The system is scalable up to three nodes, making it perfect for perimeter security. Whether you're camping or monitoring a remote site, SecureScape offers reliable security when you need it most.

System communication flow between nodes, central gadget, and mobile app

Key Features

  • Smart Threat Detection Local person detection algorithm runs on-device for real-time alerts without cloud processing.
  • Offline Operation Functions without internet, perfect for remote areas.
  • Energy Efficient Low-power design with long battery life.
  • User Alerts Receive notifications via our mobile app or central gadget.
  • Durable & Portable Designed to withstand outdoor conditions and easy to deploy.
  • Privacy Focused All processing happens on-device, no cloud services required.
